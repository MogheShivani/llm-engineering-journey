# LLM Engineering Learning Journey

This repository documents my 8-week journey to becoming an LLM Engineer.

I am learning how to build applications using Large Language Models (LLMs), including topics such as:

- Prompt Engineering
- LLM APIs
- Retrieval Augmented Generation (RAG)
- LoRA / QLoRA fine-tuning
- AI Agents
- Local vs Cloud inference
- Model evaluation

Each week includes:
- Notes explaining key concepts
- Mini projects
- Code experiments

Goal: Build practical understanding of Generative AI systems and develop portfolio-ready projects.

## Tools Used

- Python
- Jupyter Notebook
- OpenAI API
- Gemini API
- Ollama
- Cursor IDE

## Why this repo?

Learning in public helps:
- track progress
- build consistency
